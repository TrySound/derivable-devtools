# [derivable](https://github.com/ds300/derivablejs) devtools

## Usage

```js
import 'derivable-devtools'; // Should be placed before first derivable import
import { atom } from 'derivable';

const name = atom('David');

name.react(d => {
  console.log(d);
});
```
